# Linear Dashboard

Modern dashboard displaying your Linear issues with live data.

## Quick Start (Local)

1. Download `index.html`
2. Edit line 13 and add your Linear API key
3. Open in browser
4. Done!

## Features

- ✅ Live data from Linear API
- ✅ Real-time search
- ✅ Filter by status and priority
- ✅ Modern dark UI
- ✅ Responsive design
- ✅ One-click refresh
- ✅ Click to open in Linear

## Setup

### Get your Linear API key:
1. Go to Linear Settings → API
2. Create a Personal API key
3. Copy the key (starts with `lin_api_`)

### Run locally:
```bash
# Download the file
# Edit and add your API key
# Open in browser
```

## Security Note

⚠️ **IMPORTANT**: This HTML file contains your API key. Never commit it to a public repo or share it publicly. Use this only for local/private deployments.

For production use, build a proper backend API that keeps the key secure.
